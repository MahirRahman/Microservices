### Getting Started and Installation
1. Clone the repo
   ```sh
   git clone https://github.com/MahirRahman/Microservices.git
   ```
2. Install required packages using npm
   ```sh
   npm install
   ```
3. Go to terminal and run this command: 
    ``` sh
    nodemon server.js
    ```
